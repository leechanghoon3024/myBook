> 희귀동물 오픈 쇼핑몰

* 사용기술
    * server - `nginx`
    * `php`(관리자), `react-native`(앱), `typescript`, `next.js`(웹)
    * `nest.js`, `mysql`
* 설명
  ```
    1. 희귀동물 대상 오픈 쇼핑몰
    2. 셀러는 가입 신청 후 최고관리자 심사 후 승인이 되면 물품 등록
    3. 물품 등록후 최고관리자 승인 후 구매자는 물품 확인 후 구매가능
    4. 희귀동물에 대한 가이드북
    5. 최고관리자는 판매된 물품에 대해 정산
  ```
